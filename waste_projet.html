La scanette MC2200

La Zebra MC2200 est une scanette portable conçue pour la collecte de données et les applications de lecture de codes-barres dans divers environnements industriels et commerciaux. Voici un aperçu de son fonctionnement :

Lecture des codes-barres : La Zebra MC2200 est équipée d'un scanner de codes-barres intégré qui utilise la technologie de lecture laser ou d'imagerie pour capturer les données des codes-barres. Elle peut lire une variété de types de codes-barres, y compris les codes 1D et 2D. Traitement des données : Une fois qu'un code-barres est scanné, la scanette traite les données capturées et les transmet à un système de gestion des stocks, un système de point de vente (POS) ou toute autre application appropriée. Les données peuvent être transférées via une connexion sans fil (Wi-Fi ou Bluetooth) ou via un câble USB. Interface utilisateur : La Zebra MC2200 dispose d'un écran tactile ou d'un clavier intégré, selon le modèle, qui permet à l'utilisateur d'interagir avec l'appareil. L'interface utilisateur peut être utilisée pour naviguer dans les menus, saisir des données ou configurer les paramètres de l'appareil.


L'imprimante zebra ZD420

Nous avons fait le choix de cette imprimante pour différentes raison :

C'est une imprimante petite et maniable. On peut la placer un peu partout. Il nous fallait une prise RJ45 dans chaque zone où elles sont entreposées c'est pourquoi sa taille est idéale. Le fait qu'elle soit robuste en cas de chute est aussi un aspect avantageux pour nous. Elle est compatible avec différents supports ce qui permet en cas de besoin de changer le type d'étiquette. Sa configuration reste simple via un logiciel fournit avec et son installation est expliquée dans la notice. Toutes ses imprimantes zebras sont reliés à un serveur d'impression. Elles sont réparties dans chaque zone de production de l'usine nous en avons donc une dizaine.
