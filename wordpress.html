Installation wordpress avec nginx

La configuration

Grâce à cette procédure vous allez pouvoir à partir d’une Debian 11 vierge, héberger et créer votre site web WordPress via le serveur web Nginx. Il faut savoir que grâce à cette procédure nous allons utiliser mariaDB server qui sera le serveur hébergeant notre base de données, mariaDB client pour intéragir avec notre serveur, Php pour le langage qui gère le site web, et php-fpm qui gère le php pour nginx.

Nous avons besoin de certains pré requis :

  1. Se mettre en root : su -
  2. Update sa debian : apt update && apt upgrade
  3. Installer Nginx,MariaDB (server,client), PHP et PHP-FPM : Apt install nginx mariadb-server mariadb-client php php-fpm
  4. Télécharger WordPress : wget https://fr.wordpress.org/latest-fr_FR.zip


Procédure

  • Nous allons vérifier le status de nginx, il faut qu’il soit vert (actif « running ») systemctl status nginx.service
  • Nous allons nous rendre dans les fichiers du serveur web nginx cd /etc/nginx/sites-enabled
  • Ici nous avons un fichier « default », nous allons le copier à côté cp default wordpress
  • On se rend ensuite dans le fichier qui sera afficher dans le web cd /var/www/
  • On va créer un dossier qui contiendra l’application wordpress mkdir wordpress
  • On déplace le dossier wordpress que l’on a télécharger ici et on le dezipe mv wordpress /var/www/wordpress
  • tar -xvf wordpress ou unzip wordpress
  • On n’oublie pas de donner les droits en écriture à l’utilisateur www-data sur le dossier créer dans /var/www/wordpress : chown www-data:www-data -R .
